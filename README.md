# Chao Sugi Project

ESP32とAndroidアプリを組み合わせて、モーター制御やLED点灯、音声のやり取りなどを行うプロジェクトです。  
吉野杉を使ったマスコットロボ「チャオ杉ちゃん」の開発をサポートするコードを管理しています。

---

## フォルダ構成
- **ESP32/**  
  ESP32（Arduino IDE）用のコードや回路説明を配置します。

- **Android/**  
  Android Studio プロジェクトを配置します。
